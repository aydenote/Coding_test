## 22. 배수인지 확인하기

### Q. 다음 중 변수 i가 6의 배수인지 확인하는 방법으로 올바른 것은?

```
1)  i / 6 == 0
2)  i % 6 == 0
3)  i & 6 == 0
4)  i | 6 == 0
5)  i // 6 == 0
```

1. 풀이

   > 정답은 2번 입니다. i를 6으로 나눈 나머지 값이 0 이라면 6의 배수입니다.
   >
   > 1. i를 6으로 나눈 몫 == 0
   > 2. i를 6으로 나눈 나머지 == 0
   > 3. i와 6 논리곱 연산 == 0
   > 4. i와 6 논리합 연산 == 0
   > 5. 주석 처리
