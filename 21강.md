## 21. set은 어떻게 만드나요?

### Q. 다음 중 set을 만드는 방법으로 올바른 것을 모두 고르시오.

```
1)  var x = {1, 2, 3, 5, 6, 7};
2)  var x = {};
3)  var x = new Set('javascript');
4)  var x = new Set(range(5));
5)  var x = new Set();
```

1. 풀이

   > 정답은 3, 5번입니다. 3번 처럼 문자, 숫자, 객체 타입을 넣어 만들 수 있고 5번 처럼 비어 있는 set을 만들 수도 있습니다.

2. 학습
   > Set은 모든 타입의 값을 저장하는 객체자료형의 한 종류입니다. 이때, 객체 안에 값은 중복을 허용하지 않는 특징이 있습니다.
