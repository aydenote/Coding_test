## 15. 자기소개

### Q. 신학기가 시작되고, 아이들이 돌아가면서 자기소개를 하기로 했습니다.

만약 입력으로 김다정이라는 이름이 주어지면 "안녕하세요. 저는 김다정입니다."라고 출력하게
해주세요.

```
입출력

입력 : 김다정
출력 : 안녕하세요. 저는 김다정입니다.
```

1. 내 풀이 및 생각
   > 입력 받은 이름을 ES6 문법인 템플릿 리터럴(``)을 사용하여 출력하는 방식으로 풀었습니다.

```
let name = prompt("이름을 입력해주세요")

console.log(`안녕하세요 ${name}입니다.`);

```
