## 11. for를 이용한 기본 활용

### Q. 1부터 100까지 모두 더하는 Code를 <pass> 부분에 완성하세요. for를 사용해야 합니다.

```
let s = 0;

//pass

console.log(s);
```

1. 내 풀이 및 생각
   > 문제를 처음 보자마자 10번 문제랑 순서가 바뀐 것 같은데? 라는 생각이 들었습니다. 10번 문제를 풀었다면, 11번 문제는 가볍게 풀지 않을까 싶습니다. i를 1씩 증가시키며, s와 더한 값을 s에 저장하는 방식으로 풀었습니다.

```
let s = 0;

for (let i = 0; i <=100 ; i++) {
  s += i;
}

console.log(s);
```
